About Me

Hello, my name is David Horn. I am a web designer that focuses on UI Design, but I also develop small to medium size websites for individuals, companies and organizations. I live in Denver, Colorado where I spend my time as a freelance web designer most of the time, but I also enjoy hiking, lifting weights, reading and late night pizza. If you want to get in contact with me you can message me through several online platforms including twitter and email.

I am very passionate about websites and how their designs can affect their relationship with users. I can design websites and their components with a vision for wide accessibility, usability and positive user experiences. I can help you solve web design problems using prototype designs and user testing. I ensure websites are built using best-practice web standards.

About This Website

This website was built using HTML, CSS, Bootstrap and a little jQuery. I am using Jekyll for my CMS. I am using the Simple Jekyll Search scripts for my static search index. Google Fonts provides the fonts I use. All photos are provided from free, open-domain sources. This website is hosted on Github Pages.
