Toro Bailando

Website | Web Design

Toro Bailando is a website designed for a fictitional client who owns a mexican restaurant. I began by envisioning what would my client require of me for the website and its design. I also researched what restaurant's websites provided for their visitors. After an initial planning stage, I continued to sketch and illustrate a design. Then, I expanded my sketches into wireframes. After I was satisfied with my wireframes, I coded the design using HTML and CSS along with Bootstrap and jQuery. I included an interactive menu, an online contact form, custom social media icons, and a reservations form to increase interactivity between the website and the web visitor to generate business. The logo and images are from free Creative Commons licensing sources.

Tab Interface

A tab interface provides an interactive menu that lists all the various food items using JavaScript. This helps reduce the amount of scrolling visitors would need to perform to read the text content. If JavaScript is disabled in the web browser, then the text content would be displayed on a single web page without tabs.
